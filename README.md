# Eslaforde Simple Solutions

Support website for Eslaforde Simple Solutions iOS apps.

- **Simple Anagram Helper** — interactive tile workspace for anagram puzzles
- **Simple Spelling Bee Helper** — hexagonal tile workspace for spelling bee puzzles

## Hosting

Hosted on GitHub Pages at [eslaforde.com](https://eslaforde.com).

## Structure

- `/index.html` — home page (apps)
- `/support/index.html` — support & FAQ (with FAQPage structured data)
- `/privacy/index.html` — privacy policy & terms of use
- `/contact/index.html` — contact form
- `/styles.css` — shared stylesheet
- `/sitemap.xml` — lists all four pages for search engines
- `/robots.txt`, `/CNAME` — hosting configuration

## Updating

Edit the relevant HTML file (or `styles.css` for site-wide styling) and push to `main`. GitHub Pages deploys automatically.

When adding or removing pages, also update `sitemap.xml` and resubmit it in [Google Search Console](https://search.google.com/search-console).
